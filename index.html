شرح فكرة الـ SaaS في المشروع

Tenant هنا هو المطعم. عند إنشاء حساب، يتم حفظ بيانات المطعم داخل:

restaurants / userId

وتحته يتم حفظ المنيو والطلبات:

restaurants / userId / menu
restaurants / userId / orders

بذلك كل مطعم يرى بياناته فقط، وهذا يحقق مفهوم Multi-tenant SaaS.
